# Personal Site (GitHub Pages)

This folder is a standalone personal homepage template.

## Local preview

Open `/Users/lulu/Documents/New project/personal-site/index.html` directly in your browser.

## Replace images

Put your real photos in `/Users/lulu/Documents/New project/personal-site/assets/` with these names:

- `hero.jpg` (top banner landscape)
- `profile.jpg` (portrait)
- `gallery-1.jpg`
- `gallery-2.jpg`
- `gallery-3.jpg`

Then update file names in:

- `/Users/lulu/Documents/New project/personal-site/index.html`
- `/Users/lulu/Documents/New project/personal-site/styles.css` (for hero image)

## Publish on GitHub Pages

1. Create a new repo (recommended): `username.github.io`.
2. Copy all files in this folder to that repo root.
3. Push to `main` branch.
4. In GitHub repo settings, open **Pages** and set source to `Deploy from a branch` -> `main` -> `/ (root)`.
5. Wait 1-3 minutes, then open `https://username.github.io`.
